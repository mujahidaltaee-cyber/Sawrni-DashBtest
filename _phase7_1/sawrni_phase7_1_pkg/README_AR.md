# صورني Phase 7.1 — أساس تطبيق Flutter للموبايل

هذه الحزمة تنقل المشروع من واجهة ويب تجريبية إلى أساس تطبيق حقيقي يعمل على iOS و Android باستخدام Flutter.

## المحتوى

- API للموبايل داخل Laravel تحت `/api/v1/mobile/*`
- تخزين OTP بشكل مشفر Hash
- تخزين جلسات الموبايل بشكل مشفر Hash
- سجل تدقيق Mobile Audit
- دعم ملف مزود الخدمة
- واجهة Flutter:
  - اختيار الدور
  - تسجيل دخول برقم الهاتف
  - إدخال OTP
  - واجهة العميل
  - واجهة مزود الخدمة
  - إرسال ملف المزود للمراجعة

## قواعد الإنتاج المهمة

- مزود الخدمة لا يظهر للعملاء قبل موافقة الإدارة/COO.
- قائمة العميل تعرض فقط المزودين الموافق عليهم وغير المعلقين.
- معلومات التواصل الحساسة لا يتم عرضها للعميل.
- OTP لا يخزن كنص صريح.
- Token الجلسة لا يخزن كنص صريح.
- رمز التطوير 123456 يظهر فقط خارج بيئة الإنتاج.
