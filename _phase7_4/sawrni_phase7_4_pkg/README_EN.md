# Sawrni Phase 7.4 — Provider Mobile Dashboard

This phase adds a real provider-facing mobile feature set:

- Incoming booking list.
- Booking details screen.
- Accept booking.
- Reject booking with internal reason.
- Sensitive customer contact masking before confirmed booking.
- Sawrni brand and slogan: "استوديو كامل بجيبك".
- Laravel mobile provider API routes when the backend exists.

This is not a test-only phase. It is production source work for the Flutter mobile app.
