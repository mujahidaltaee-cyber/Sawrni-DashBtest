# صورني Phase 7.5

هذه المرحلة تضيف منطق دورة الحجز والعربون داخل تطبيق صورني.

الشعار المعتمد:
استوديو كامل بجيبك

الوظائف:
- طلب حجز من العميل
- قبول أو رفض المزود
- حالة انتظار العربون
- تأكيد الحجز بعد العربون
- إخفاء بيانات التواصل قبل التأكيد
- عمولة المنصة 15%
- تعديل العميل لمدة 3 ساعات بعد العربون
- إلغاء مع استرجاع العربون خلال ساعة واحدة فقط
- بعد ساعة يصبح العربون غير قابل للاسترجاع

هذه مرحلة إنتاجية وليست مرحلة اختبار فقط.
