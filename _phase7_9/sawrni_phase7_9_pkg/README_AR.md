# صورني Phase 7.9 — تجهيز بناء APK للأندرويد

هذه المرحلة تجهز تطبيق Flutter الحقيقي حتى يتحول لاحقاً إلى APK قابل للتثبيت على الهاتف.

الشعار:

**استوديو كامل بجيبك**

## ماذا تضيف؟

- إعدادات API base URL عن طريق `--dart-define`.
- سكربت بناء APK debug.
- سكربت Windows للبناء.
- تجهيز صلاحيات Android الأساسية.
- تعليمات واضحة لمكان ملف APK بعد البناء.
- تجهيز مجلد الهوية البصرية للأيقونة والـ splash.

## ملاحظة مهمة

Codespaces غالباً لا يحتوي Android emulator ولا Flutter SDK، لذلك هذه المرحلة تجهز الكود فقط. بناء وتجربة التطبيق الحقيقي يكون على Windows مع Android Studio أو بيئة فيها Flutter.
