# صورني Phase 7.8 — أساس الإشعارات

هذه المرحلة تضيف أساس الإشعارات لتطبيق صورني على الموبايل والباكند.

تشمل:
- مركز إشعارات للزبون.
- مركز إشعارات لمزود الخدمة.
- موديلات إشعارات داخل Flutter.
- جداول وواجهات API للإشعارات داخل Laravel عند وجود الباكند.
- إشعارات الحجز، العربون، التسليم، التقييم، موافقة المزود، وتنبيه الدين.
- الهوية العربية أولاً مع الشعار: استوديو كامل بجيبك.

هذه الحزمة موجهة للكود الإنتاجي وليست لاختبارات فقط.
