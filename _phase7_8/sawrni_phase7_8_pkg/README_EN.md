# Sawrni Phase 7.8 — Notifications Foundation

This phase adds the mobile and backend notification foundation for Sawrni.

Includes:
- Customer notification center.
- Provider notification center.
- Notification models and local demo provider for Flutter.
- Laravel notification tables and API routes when backend exists.
- Notification types for booking, deposit, delivery, reviews, provider approval, and debt warnings.
- Arabic-first Sawrni identity with the slogan: استوديو كامل بجيبك.

This package is production-source oriented. It does not add local-login or test-only bypass routes.
