# صورني Phase 7.7 — التقييمات والموافقة قبل النشر

هذه المرحلة تضيف منطق التقييمات في التطبيق الحقيقي:

- العميل يرسل تقييم الخدمة.
- التقييم يبقى بحالة `pending_review`.
- لا يظهر التقييم للعامة إلا بعد موافقة الإدارة/COO/السوبر أدمن.
- التقييم المرفوض يبقى داخلياً للأرشفة ولا يظهر للعامة.
- تقييم مزود الخدمة العام يجب أن يُحسب فقط من التقييمات الموافق عليها.
- كل قرار موافقة أو رفض يُسجل في audit logs.

الشعار المعتمد:

> استوديو كامل بجيبك
